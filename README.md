This document provides best practices and implementation guidelines for optimizing your website for search engines (SEO).
It covers on-page, technical, and content SEO strategies designed for modern frameworks like **Astro.js**, **Next.js**, and CMS integrations (e.g. **Payload CMS**).

---

## 🧭 Overview

SEO (Search Engine Optimization) improves your website’s visibility on search engines like Google, Bing, and Yahoo.
A well-optimized site helps increase organic traffic, ranking, and conversion rates.

---

## 🚀 Key SEO Components

### 1. Meta Tags

Add meta tags in your page `<head>` section or via your framework’s layout component.

```html
<title>Your Page Title</title>
<meta name="description" content="A concise, keyword-rich description of your page.">
<meta name="keywords" content="primary keyword, secondary keyword, brand name">
<meta name="robots" content="index, follow">
<meta property="og:title" content="Your Page Title">
<meta property="og:description" content="Description for social media.">
<meta property="og:image" content="/path/to/image.jpg">
<meta name="twitter:card" content="summary_large_image">
```

關鍵字（Keyword）為搜尋引擎用來查詢資訊、索引文章的專有名詞，而對於使用者來說，關鍵字意思也就是輸入在搜尋引擎上的任何搜尋詞。

隨著網路的普及，搜尋引擎上的任何行為、字詞，都有了SEO領域的專有名詞，而每次只要提到SEO，「關鍵字」就時常被提出來討論與研究，如果要論其重要性必須從內容營銷去探討：
以Google搜尋引擎為例，它就是利用各種關鍵字的組合搭配去判斷用戶的搜尋意圖，接著，盡可能給予最符合需求的搜尋結果，換言之，[ERP上不管是文章、圖片、影音內容等，都和關鍵字有密不可分的關係！

### 【關鍵字2大類】

關鍵字通常還可以細分為2大類：

* ​**核心關鍵字**​：也俗稱為「大字」，指涵蓋層面廣、搜尋量大、流量大的關鍵字，且通常核心關鍵字競爭程度較高。
* ​**長尾關鍵字**​：由核心關鍵字延伸出來的字詞，通常字數較多，可能是各種詞組組合或是問句等，雖然搜尋量、流量沒有核心關鍵字大，但是是搜尋意圖更明確的關鍵字，因此搜尋引擎提供給用戶的內容也會更精準。

#### 關鍵字行銷的好處有哪些？
